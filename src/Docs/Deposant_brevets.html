nr_famille_docdb ==> Numéro de la famille DOCDB; au sein d’une famille DOCDB, les différentes demandes partagent exactement les mêmes priorités

nom_demandeur ==> Nom standardisé du déposant tel qu’enregistré dans la base de données de l’Office européen des brevets

code_pays ==> Code pays du déposant

siren ==> siren
